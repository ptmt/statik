---
title: html.footnotes.display
type: string
required: false
default: LIST
category: html
order: 1
---

Controls how Markdown footnotes are rendered in the final HTML. Use `LIST` (default) to keep the generated `<div class="footnotes">` block at the bottom of the article. Switch to `HOVER` to hide the list and replace each `[^1]` reference with an inline tooltip that shows the footnote text on hover/focus.
